1 两数之和

https://leetcode.cn/problems/two-sum/



算法：

因为题目中提到每种输入只会对应一种输出，所以利用map存储已经遍历过的元素和下标，达到O(1)查找。

```go
// date: 2020-09-14
func twoSum(nums []int, target int) []int {
    set := make(map[int]int, len(nums))
    res := []int{}
    for j, v := range nums {
        d := target - v
        if i, ok := set[d]; ok && i != j {
            res = append(res, i, j)
            break
        } else {
            set[v] = j
        }
    }
    return res
}
```
