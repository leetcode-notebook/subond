27 移除元素

https://leetcode.cn/problems/remove-element/

题目要求：原地移除数组中等于val的元素，并返回数组的新长度。

算法：前后指针

```go
// date 2022-09-15
func removeDuplicates(nums []int) int {
    idx := 0
    for i := 0; i < len(nums); i++ {
        if i > 0 && nums[i] == nums[i-1] {
            continue
        }
        nums[idx] = nums[i]
        idx++
    }
    return idx
}
```
