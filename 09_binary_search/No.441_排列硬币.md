## 441 排列硬币-简单

题目：

你总共有 `n` 枚硬币，并计划将它们按阶梯状排列。对于一个由 `k` 行组成的阶梯，其第 `i` 行必须正好有 `i` 枚硬币。阶梯的最后一行 **可能** 是不完整的。

给你一个数字 `n` ，计算并返回可形成 **完整阶梯行** 的总行数。



分析：

直接计算。

```go
// date 2023/12/06
func arrangeCoins(n int) int {
    ans := 0
    sum := 0

    for sum <= n {
        ans++
        sum += ans
    }

    return ans-1
}
```
