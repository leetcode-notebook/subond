## 509 斐波那契数

题目：

斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
给定 n ，请计算 F(n) 。


分析：

动归，自底向上，也就是递推。

```go
// date 2023/11/11
func fib(n int) int {
    if n < 2 {
        return n
    }
    f1, f2 := 0, 1
    fn := 0
    for i := 2; i <= n; i++ {
        fn = f1 + f2
        f1, f2 = f2, fn
    }
    return fn
}
```
